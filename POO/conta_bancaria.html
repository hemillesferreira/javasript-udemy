Criar uma classe ContaBancaria que tenha uma propriendade privada para o saldo

Utilizar getters e setters para acessar e atualizar o saldo.O saldo só pode ser atualizado atraves de depositos e saques, e nao deve permitir que o saldo se torne negativo.

Implementar metodos para depoisito (depositar(valor)) e saque (sacar(valor)), onde o saque so é permitido se houver saldo sufuciete.
